Quick reference for common actions in JUSTUS.

Action How to do it
View profile Long-press (hold) on any chat in the list
Mute notifications In profile, tap the mute toggle
Clear chat In profile, tap "Clear Chat"
Update status Go to Updates tab → tap "Add Status Update"
Make a call Go to Calls tab → tap the phone icon
Change theme Settings tab → pick Light / Dark / OLED
Back to chats Tap the ← arrow in top left
💡 Tip On mobile, long-press means pressing and holding a chat for about one second until the profile opens.
